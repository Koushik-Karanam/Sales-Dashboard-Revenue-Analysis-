# Sales-Dashboard-Revenue-Analysis-
Built an interactive Tableau dashboard connected to PostgreSQL, visualizing monthly revenue trends, top products, and regional sales across 8 markets. Automated reporting via Python (Pandas) &amp; SQL with daily data refresh. Pareto analysis identified top 20% products driving 80% revenue. Result: 15% efficiency gain.

Features and Applications: 

Designed and built interactive sales dashboard in Tableau connecting to PostgreSQL database, visualizing monthly 
revenue trends, top-performing products by sales volume, and regional sales distribution across 8 geographic 
markets. 
• Created automated monthly revenue reports using SQL queries and Python scripts; dashboards updated daily with 
fresh data enabling real-time monitoring of sales performance against targets. 
• Analyzed historical sales data (2+ years) to identify seasonal trends, product lifecycle patterns, and revenue growth 
opportunities; presented findings to executive team for strategic planning. 
• Identified top 20% performing products generating 80% of revenue using Pareto analysis; insights enabled 
management to optimize inventory allocation and focus marketing efforts on high-ROI products. 
• Tech Stack: Tableau, PostgreSQL, Python (Pandas), SQL | Result: 15% increase in sales team efficiency and 
improved data-driven decision making at management level
